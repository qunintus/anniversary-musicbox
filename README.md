## deploy
部署到你个人的github上
```
git clone https://github.com/seasonstar/anniversary.git

cd anniversary

git branch gh-pages

git checkout gh-pages

git push origin gh-pages
```

然后打开`http://<自己的账号名>.github.io/anniversary/`，就可以看到了。
